Product Overview
系统总览
SSLAB 是专为高校实验室设计的综合管理平台，通过统一的 Web 界面，集中控制实验室内所有智能设备，大幅降低实验准备时间，提升教学安全性。
⚡
教学电源管理
低压/高压/大电流多路电源实时监控与控制，学生分组独立管理
🌡️
环境监测与控制
温湿度、天气数据实时采集，联动供水排风、灯光与变频器
📋
互动教学答题
全班学生实时抢答，教师可视化统计答题结果与作答速度
🖥️
升降台与电脑
实验台升降与电脑电源一键控制，0-16 台设备批量管理
主界面采用卡片式布局，所有功能模块一目了然；顶部标签快速切换四大功能区，右上角实时显示系统在线状态。
主界面
  • 全 Web 架构 — 无需安装 App，浏览器直接访问
  • 实时双向通信 — WebSocket 推送，状态毫秒级同步
  • 多协议兼容 — Modbus RTU/TCP、自研 JSON 协议
  • 仿真测试模式 — 无需接硬件即可演示全部功能

一套系统，全面覆盖

从实验台电源到环境传感器，从学生互动到灯光控制，SSLAB 用单一界面替代多个独立操控盒。

01
Feature Module
教学电源管理
低压、高压、大电流三路教学电源统一管控，学生分组供电精确到每台实验台。
左侧低压电源实时显示电压 / 电流；中列高压与大电流开关附带延时倒计时保护；右侧学生分组 A/B/C/D 独立控制。
教学电源
低压直流输出范围
0 — 30 V / 2 A
高压交流输出
240 — 300 V
学生分组数
A / B / C / D 四组
通信协议
Modbus RTU over TCP
  • 实时监测 — 电压、电流数值 1 秒刷新，直观蓝字高亮显示
  • 学生电源同步 — 勾选同步后，学生台电压自动跟随教师设定值
  • 延时保护 — 高压与大电流开启前内置 10 s 倒计时提醒
  • 已连接状态 — 绿色徽章实时标识每路电源的网络连接状态
02
Feature Module
实验环境管理
多传感器环境监测 + 供水排风 + 灯光照明 + 变频器转速，四大子系统统一界面一屏掌控。
左侧环境监测数据卡显示天气、温湿度、风速、气压及用电量；右侧四个控制卡分别管理供水/排风、升降台、灯光和变频器。
实验环境
  • 环境监测 — 温度、湿度、体感温度、能见度、气压、风速实时采集
  • 供排水控制 — 实验台供水阀 / 排风扇开关一键切换
  • 灯光分区 — 黑板灯与教室灯独立控制，支持定时联动
  • 升降台 — 最多 16 台电动实验台升降同步控制
  • 变频器 — 教学电机转速三档调节（低 / 中 / 高速）
传感器接入
USB / 网络型
控制设备数
最多 16 台升降台
继电器通道
10 路 (USR-IO808)
03
Feature Module
互动教学管理
全班学生实时抢答，教师即时掌握正确率、最快作答者，让课堂互动数字化、可视化。
A1—D4 共 16 个学生座位答题框实时更新，正确答案绿色高亮，错误红色标注，底部汇总统计正确人数、错误人数及最快答题者。
互动答题
  • 16 座位矩阵 — A/B/C/D 组 × 4 排，覆盖主流实验室配置
  • 实时推送 — 学生答题后 < 1 s 在教师端显示
  • 答题倒计时 — 可配置作答时限，超时自动锁定
  • 结果统计 — 自动统计正确 / 错误人数与最快答题学生
  • 空气质量联动 — 同步显示甲醛 / TVOC 浓度辅助教学
